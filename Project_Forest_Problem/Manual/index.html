実行方法（実行ファイルの動かし方）

実行するためのコマンドは以下の通りである。

make treetest

tree.txtのファイルを読み込む。

make foresttest

forest.txtのファイルを読み込む。

make semilatticetest

semilattice.txtのファイルを読み込む。
